1""" 

2:mod:`cape.pyover.options.overnml`: OVERFLOW namelist options 

3============================================================== 

4 

5This module provides a class to mirror the Fortran namelist capability. The 

6module provides a class, :class:`pyOver.options.overnml.OverNml`, which 

7interprets the settings of the ``"Overflow"`` section of the master JSON file. 

8These settings are then applied to the main OVERFLOW input file, the 

9``overflow.inp`` namelist. 

10 

11At this time, nonunique section names are not allowed. To modify parameters in 

12repeated sections, use the options in the ``"Grids"`` section using 

13:class:`pyOver.options.gridSystem.GridSystem`. 

14 

15:See also: 

16 * :mod:`cape.pyover.options.gridSystem` 

17 * :mod:`cape.pyover.overNamelist` 

18 * :mod:`cape.pyover.cntl` 

19 * :mod:`cape.namelist2` 

20""" 

21

33 :Call:

35 :Inputs: 

36 *opts*: :class:`pyOver.options.Options` 

37 Options interface 

38 *i*: :class:`int` or ``None`` 

39 Run sequence index

42 GLOBAL namelist 

43 :Versions: 

44 * 2016-02-01 ``@ddalle``: First version 

45 """ 

46 # Get the value 

47 d = getel(self.get('GLOBAL'), i) 

48 # Check for None

83 # Reduce to a single run sequence 

84 def select_namelist(self, i=0): 

85 """Reduce namelist options to a single instance (i.e. sample lists) 

86  

87 :Call: 

88 >>> d = opts.select_namelist(i) 

89 :Inputs: 

90 *opts*: :class:`pyOver.options.Options` 

91 Options interface 

92 *i*: :class:`int` or ``None`` 

93 Phase number 

94 :Outputs: 

95 *d*: :class:`pyOver.options.odict` 

96 Project namelist 

97 :Versions: 

98 * 2015-10-18 ``@ddalle``: First version 

99 * 2016-02-01 ``@ddalle``: Copied from pyFun 

100 """ 

101 # Initialize output 

102 d = {} 

103 # Loop through keys 

104 for sec in self: 

105 # Get the list 

106 L = getel(self[sec], i) 

107 # Initialize this list. 

108 d[sec] = {} 

109 # Loop through subkeys 

110 for k in L: 

111 # Select the key and assign it. 

112 d[sec][k] = getel(L[k], i) 

113 # Output 

114 return d

116 # Get value by name 

117 def get_namelist_var(self, sec, key, i=None): 

118 """Select a namelist key from a specified section 

119  

120 Roughly, this returns ``opts[sec][key]``. 

121  

122 :Call: 

123 >>> val = opts.get_namelist_var(sec, key, i=None) 

124 :Inputs: 

125 *opts*: :class:`pyOver.options.Options` 

126 Options interface 

127 *sec*: :class:`str` 

128 Section name 

129 *key*: :class:`str` 

130 Variable name 

131 *i*: :class:`int` or ``None`` 

132 Phase number 

133 :Outputs: 

134 *val*: :class:`int` | :class:`float` | :class:`str` | :class:`list` 

135 Value from JSON options 

136 :Versions: 

137 * 2015-10-19 ``@ddalle``: First version 

138 * 2016-02-01 ``@ddalle``: Copied from pyFun 

139 """ 

140 # Check for namelist 

141 if sec not in self: return None 

142 # Select the namelist 

143 d = getel(self.get(sec), i) 

144 # Select the value. 

145 return getel(d.get(key), i)
