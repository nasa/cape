2r""" 

3:mod:`cape.units`: Unit Conversion Module 

4========================================== 

5 

6CAPE includes this unit-conversion module to interpret various units for 

7inputs and post-processing. Many CFD solvers, and some trajectory tools 

8that are used to provide flight conditions, expect imperial units, which 

9can be challenging to interpret and difficult to use in complications. 

10 

11The main function of this module is called :func:`mks`, which takes a 

12string as input and tries to convert it to a numerical output as 

13multiple of meters, kilograms, and seconds (and Kelvins and amperes or 

14several other fundamental units as needed). For example it will return 

150.0254 if the input is ``"inch"``, which is the length of an inch in 

16 meters. 

17 

18 .. code-block:: pycon 

19  

20 >>> mks("inch") 

21 0.0254 

22 

23This function also takes more complexly defined units using exponents, 

24multiples and ratios, and SI prefixes. For example, this converts 

25thousand pounds per square foot to the appropriate mks unit, which is 

26Pascals. 

27 

28 .. code-block:: pycon 

29  

30 >>> mks("klbf/ft^2") 

31 4.7880e+04 

32 

33The module also contains several simple :class:`float` values, such as 

34*psf* for the value of pounds per square feet in Pascals. There is a 

35:class:`dict` called *units_dict* that contains a full dictionary of 

36recognized unit names. 

37 

38 .. code-block:: pycon 

39  

40 >>> units_dict['psf'] 

41 47.88025903135141 

42 

43"""

251def mks(unit_lbl): 

252 r"""Convert a string for arbitrary nits to numerical MKS value 

253  

254 :Call: 

255 >>> val = mks(unit_lbl) 

256 :Examples: 

257 >>> mks("kg") 

258 1.0 

259 >>> mks("lbf/ft^2") 

260 47.880 

261 :Inputs: 

262 *unit_lbl*: :class:`str` | :class:`unicode` 

263 String describing units 

264 :Outputs: 

265 *val*: :class:`float` 

266 Units converted to numerical value assuming MKS system 

267 :Versions: 

268 * 2018-04-12 ``@dalle``: Version 1.0 

269 """ 

270 # Check for null input 

271 if unit_lbl is None: 

272 return 1.0 

273 # Localize input 

274 lbl = unicode(unit_lbl) 

275 # Replace exponent characters 

276 lbl = re.sub(u'¹', '', lbl) 

277 lbl = re.sub(u'²', '^2', lbl) 

278 lbl = re.sub(u'³', '^3', lbl) 

279 # Degree sign 

280 lbl = re.sub(u'°', " deg", lbl) 

281 # Replace fractions 

282 lbl = re.sub(u'½', '0.50', lbl) 

283 lbl = re.sub(u'¼', '0.25', lbl) 

284 lbl = re.sub(u'¾', '0.75', lbl) 

285 # Replace multiplication symbols with a space 

286 lbl = re.sub(u'[\\*×]', ' ', lbl) 

287 # Substitute division signs (unlikely, right?) 

288 lbl = re.sub(r'÷', '/', lbl) 

289 # Replace double asterisk with '^' 

290 lbl = re.sub(r'\*\*', '^', lbl) 

291 

292 # Remove extra spaces around '^' symbols 

293 lbl = re.sub(r'\s*\/+\s*', '/', lbl) 

294 # Place a space in front of each slash 

295 lbl = re.sub(r'\/', ' /', lbl) 

296 

297 # Initialize the output value 

298 val = 1.0 

299 # Split off each chunk in the form '[/][prefix](unit)[^exponent]' 

300 lbls = [lbli.strip() for lbli in lbl.split()] 

301 # Loop through labels 

302 for lbli in lbls: 

303 # Check for an exponent character 

304 if '^' in lbli: 

305 # Split on exponent 

306 lbli, expi = lbli.split('^') 

307 else: 

308 # No exponent 

309 expi = '1' 

310 # Evaluate exponent 

311 powi = float(expi) 

312 

313 # Check if this is a denominator 

314 if lbli.startswith('/'): 

315 # Denominator 

316 powi = -powi 

317 # Strip leading slash 

318 lbli = lbli[1:] 

319 

320 # Check for numeric values (does not allow '2km', 'km2', etc.) 

321 try: 

322 # Try converting whole symbol to number 

323 vali = float(lbli) 

324 # If that worked, apply the exponent and move on 

325 val *= (vali**powi) 

326 continue 

327 except ValueError: 

328 pass 

329 

330 # Initialize current version of symbol 

331 cur = lbli 

332 pfx = '' 

333 # Loop until the current symbol (w/o prefix) matches list 

334 while cur and (cur not in units_dict): 

335 # Move the first character into the prefix 

336 pfx = pfx + cur[0] 

337 cur = cur[1:] 

338 

339 # Check for valid match 

340 if cur in units_dict: 

341 # Get the value 

342 vali = units_dict[cur] 

343 else: 

344 # Unrecognized 

345 print(Warning("Symbol '%s' is not a recognized unit" % lbli)) 

346 # Assume 1.0 

347 vali = 1.0 

348 

349 # Move prefix to "remaining prefix" 

350 rpf = pfx 

351 # Loop through characters of prefix 

352 while len(rpf) > 0: 

353 # Current prefix 

354 cpf = rpf 

355 # Remaining part of prefix 

356 rpf = '' 

357 # Look for matches 

358 while (len(cpf) > 1) and (cpf not in prefix_name): 

359 # Move the last character out of the current prefix 

360 rpf = cpf[-1] + rpf 

361 cpf = cpf[:-1] 

362 # Process prefix 

363 if cpf in prefix_name: 

364 # Multiply this value into the value of the current symbol 

365 vali *= prefix_dict[cpf] 

366 else: 

367 # Unrecognized 

368 print(Warning("Prefix '%s' was not recognized" % cpf)) 

369 

370 # Multiply the final value 

371 val = val * vali**powi 

372 

373 # output 

374 return val
